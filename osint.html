Useful Cyber Links

Please find a collection of useful cyber security resources and links that I have found helpful in my career. I will continue to update this page as I find more resources that I think would be helpful to others in the industry. I've started this list with OSINT links that I use daily to check varioyus IPs, domains, and hashes (Indicators of Compromise, or IoC's) when triaging security incidents. Feel free to reach out if you have any suggestions for resources to add!


AbuseIPDB

AbuseIPDB is a tool used to report IP address's associated with malicious activity, or check to see if an IP address has been reported, by using this tool.
I use this tool to check the reputations of IoC's, and what they may have bene reported for in the past.

Link to Tool


Shodan

Shodan is a search engine for Internet-connected devices. You can use this to search for IP addresses and see what devices / domains are hosted on it, as well as what open ports the relevant web servers have open, and OS level information.
I use this tool to get more information about IoC's, and what they are used for.

Link to Tool


VirusTotal

VirusTotal is a tool that allows you to upload files or paste URLs for analysis by multiple antivirus engines and other security tools.
I use this tool to analyze suspicious files and URLs that are associated with IoC's.

Link to Tool
