Useful Cyber Links

Please find a collection of useful cyber security resources and links that I have found helpful in my career. I will continue to update this page as I find more resources that I think would be helpful to others in the industry.

I've started this list with OSINT links that I use daily to check various IPs, domains, and hashes (Indicators of Compromise, or IoC's) when triaging security incidents.
There's also some good free learning resources I've added links to.

Feel free to reach out if you have any suggestions for resources to add!


AbuseIPDB - OSINT Tool

AbuseIPDB is a tool used to report IP address's associated with malicious activity, or check to see if an IP address has been reported, by using this tool.
I use this tool to check the reputations of IoC's, and what they may have bene reported for in the past.

Link to Tool


Shodan - OSINT Tool

Shodan is a search engine for Internet-connected devices. You can use this to search for IP addresses and see what devices / domains are hosted on it, as well as what open ports the relevant web servers have open, and OS level information.
I use this tool to get more information about IoC's, and what they are used for.

Link to Tool


VirusTotal - OSINT Tool

VirusTotal is a tool that allows you to upload files or paste URLs for analysis by multiple antivirus engines and other security tools.
I use this tool to analyze suspicious files and URLs that are associated with IoC's.

Link to Tool


CyberChef - OSINT Tool

CyberChef is a simple, intuitive web app built by GCHQfor analysing and decoding data without having to deal with complex tools or programming languages. CyberChef encourages both technical and non-technical people to explore data formats, encryption and compression.
I use this tool to decode and analyze data, as well as defang / sterilize IoC's when sharing them with others.

Link to Tool


Palo Alto's Applipedia - OSINT Tool

The Applipedia is the application database that Palo Alto Networks uses along with App-ID to identify applications traveling through Palo Alto Networks firewalls.
As I work with a Palo Alto Networks firewall, I use this tool to get more information about applications that are being used in my environment, and to help with triaging security incidents. It's also useful for learning about new applications that I may not be familiar with, and how they work, whether you see them in your environment or not.

Link to Tool


Browserling - OSINT Tool

Browserling is an online, interactive cross-browser testing service that lets developers and testers run real browsers (Chrome, Firefox, Edge, Safari, Opera, Internet Explorer) in a secure, virtualized, and sandboxed environment without requiring installations.
I use Browserling to review potentially malicious websites in a safe environment, without having to install or run the website inside my own browser.

Link to Tool


Cisco Networking Academy (NetAcad) - Free Learning Resource

Learn from expert educators around the globe, both in-person and online, for free. Start with individual courses or dive right in to a career path. The learning choices and your tech career future are yours for the taking.
I've done a few different Cyber Security and Networking courses on this platform which I'd definetly recomend to anyone looking to learn more about those topics (see my Certifications page).
